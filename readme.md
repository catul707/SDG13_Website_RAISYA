# Climate Action Hackathon 2025 — University of Cyberjaya

A responsive, accessible, and mobile-first website designed for the **Climate Action Hackathon 2025** hosted by the **University of Cyberjaya**.  
The site provides event details, schedules, mentors, tracks, registration forms, and sponsor information — focused on **UN SDG 13: Climate Action**.

## Overview

This website serves as an official event page for the **Climate Action Hackathon 2025**, providing participants, mentors, and partners with all essential information — from the event overview to the registration form.

**Event Theme:** *"Building practical solutions that reduce climate risks"*  
**Focus:** *Sustainable Development Goal (SDG) 13 — Climate Action*  
**Date:** 9–11 November 2025  
**Venue:** University of Cyberjaya

Responsive and mobile-first layout
